О нас Документация Войти

Регистрация и почта на вашем домене — без сервера и без кода

Добавьте авторизацию, управление пользователями и корпоративную почту под своим доменом. Всё через REST API. Работает за 5 минут.

Попробовать бесплатно

Кому это нужно:

  • Владельцам сайтов, которым нужен контроль над пользователями
  • SaaS-сервисам и стартапам, которые не хотят писать свою auth-систему
  • Тем, кто хочет email-ящики @своёдоменноеимя

Что вы получаете:

1. Регистрация и вход на вашем сайте

REST API для регистрации, логина, восстановления пароля, редактирования профиля.

2. Пользователи под контролем

Создавайте пользователей вручную или через API. Настраивайте свойства: имя, ник, телефон, что угодно.

3. Email на вашем домене

Корпоративные ящики @вашдомен.com. Почта работает через IMAP/SMTP, доступна в любом почтовом клиенте.

4. Без серверной части

Вся инфраструктура на нашей стороне. Вам нужно только настроить DNS.

5. Гибкая интеграция

Пишите свои страницы регистрации/входа или закажите у нас. Все возможности доступны через API.

Как это работает:

  1. Зарегистрируйтесь на userid.sh

  2. Добавьте свой домен

  3. Настройте DNS-записи

  4. Укажите поля профиля

  5. Интегрируйте API на сайте

  6. Всё работает!
